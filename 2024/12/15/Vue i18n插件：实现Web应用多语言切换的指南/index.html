Vue i18n：一款实现应用多语言切换的插件

vue
vue
发布日期:   2024-12-15

vue i18n 是 Vue.js 官方的国际化插件，i18n 是 “internationalization” 的缩写（首字母 i + 中间 18 个字母 + 尾字母 n），意思是“国际化”。它指的是让应用程序支持多种语言的能力。

安装 vue i18n

npm install vue-i18n --save
# 或者
yarn add vue-i18n

使用步骤

1. 创建 i18n 实例

通常我们会将 i18n 配置单独放在一个文件中，比如 src/locales/index.js。

import { createI18n } from 'vue-i18n'
// 导入语言包
import en from './langs/en.json';
import zh from './langs/zh.json';
// 定义语言包
const messages = {
  en,
  zh
}
// 导出实例
export default createI18n({
  legacy: false, // 使用 composition API 模式
  locale: 'zh',  // 默认语言
  fallbackLocale: 'en', // 回退语言
  globalInjection: true, // 全局注入 $t 方法
  messages
})

2. 设置语言包，主要工作量在此

// en.json
{
  "message": {
    "hello": "hello world"
  }
}

// zh.json
{
  "message": {
    "hello": "你好，世界"
  }
}

3. 在 main.js 或者 main.ts 中注册 i18n 插件

import App from './App.vue'
import i18n from './i18n' // 引入 i18n 实例
new Vue({
  el: '#app',
  i18n,
})
// vue3:  createApp(App).use(i18n).mount('#app')

4. 在组件中使用翻译内容

模板中直接使用 $t

<template>
  <div>{{ $t('message.hello') }}</div>
</template>

Vue i18n 还提供了 v-t 指令，可以直接在模板中进行翻译：

<template>
  <div v-t="'message.hello'"></div>
</template>

5. 切换语言

在 Vue2 组件中，可以通过修改 i18n.locale 属性来切换语言

<template>
  <div>
    <button @click="changeLanguage('en')">English</button>
    <button @click="changeLanguage('zh')">中文</button>
  </div>
</template>

<script>
export default {
  methods: {
    changeLanguage(lang) {
      this.$i18n.locale = lang;
      localStorage.setItem('locale', lang); // 本地化存储用户选择的语言
    }
  }
}
</script>

在 vue3 组件里通过 useI18n() 的 locale 属性来切换语言

<template>
  <button @click="changeLanguage('zh')">中文</button>
  <button @click="changeLanguage('en')">English</button>
</template>

<script setup>
import { useI18n } from 'vue-i18n'

const { locale } = useI18n()

function changeLanguage(lang) {
  locale.value = lang
}
</script>

常见问题

$t is not defined

确保在 i18n.js 中设置了 globalInjection: true

设置的语言如何本地化存储

// 将用户选择的语言偏好保存在本地存储中
methods: {
  changeLanguage(lang) {
    this.$i18n.locale = lang;
    localStorage.setItem('locale', lang); // 保存用户选择的语言
  }
}

// 并在应用初始化时读取本地存储
const savedLocale = localStorage.getItem('locale') || 'en';
const i18n = createI18n({
  // ...
  locale: savedLocale,
});

当组件需要根据条件动态加载时，如何在组件初始化时能够正确地获取翻译信息

export default {
  name: 'DynamicComponent',
  created() {
    this.setLanguage(this.$i18n.locale);
  },
  methods: {
    setLanguage(lang) {
      // 根据语言设置组件的翻译
      this.$i18n.locale = lang;
    }
  }
}

如何支持日期、数字格式化？

const messages = {
  en: {
    date: 'MM/DD/YYYY',
    time: 'h:mm A'
  },
  zh: {
    date: 'YYYY/MM/DD',
    time: 'HH:mm'
  }
};

并在组件中使用：
<template>
  <div>
    {{ $d(new Date(), 'date') }}
    {{ $t('time') }}
  </div>
</template>
