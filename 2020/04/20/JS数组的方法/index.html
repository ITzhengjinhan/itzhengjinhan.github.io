JS数组常用的方法

数组
js
发布日期:   2020-04-20

介绍：
整理目前所用过的数组方法，学习新增的 es6 方法。

1：arr.push() 从后面添加元素，返回值为添加完后的数组的长度

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.push(5))   // 6
console.log(arr) // [1,2,3,4,5,5]

2：arr.pop() 从后面删除元素，只能是一个，返回值是删除的元素

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.pop())     // 5
console.log(arr)  //[1,2,3,4]

3：arr.shift() 从前面删除元素，只能删除一个 返回值是删除的元素

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.shift())  // 1
console.log(arr)   // [2,3,4,5]

4：arr.unshift() 从前面添加元素, 返回值是添加完后的数组的长度

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.unshift(2))    // 6
console.log(arr)  //[2,1,2,3,4,5]

5：arr.splice(i,n) 删除从 i(索引值)开始之后的那个元素。返回值是删除的元素参数： i 索引值 n 个数

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.splice(2,2))     //[3,4]
console.log(arr)    // [1,2,5]

6：arr.concat() 连接两个数组 返回值为连接后的新数组

是否改变原数组：否

let arr = [1,2,3,4,5]
console.log(arr.concat([1,2]))  // [1,2,3,4,5,1,2]
console.log(arr)   // [1,2,3,4,5]

7：join(),将数组变为字符串

是否改变原数组：否

var arr = ["George","John","Thomas"]
console.log(arr.join())//George,John,Thomas
console.log(arr.join("."))//George.John.Thomas

8：arr.sort() 将数组进行排序,返回值是排好的数组，默认是按照最左边的数字进行排序，不是按照数字大小排序的，见例子。

是否改变原数组：是

let arr = [2,10,6,1,4,22,3]
console.log(arr.sort())   // [1, 10, 2, 22, 3, 4, 6]
let arr1 = arr.sort((a, b) =>a - b)
console.log(arr1)   // [1, 2, 3, 4, 6, 10, 22]
console.log(arr)   // [1, 2, 3, 4, 6, 10, 22]

let arr2 = arr.sort((a, b) =>b-a)
console.log(arr2)  // [22, 10, 6, 4, 3, 2, 1]
console.log(arr)  // [22, 10, 6, 4, 3, 2, 1]

9：arr.reverse() 将数组反转,返回值是反转后的数组

是否改变原数组：是

let arr = [1,2,3,4,5]
console.log(arr.reverse())    // [5,4,3,2,1]
console.log(arr)    // [5,4,3,2,1]

10：arr.slice(start,end) 切去索引值 start 到索引值 end 的数组，不包含 end 索引的值（左闭右开），返回值是切出来的数组

是否改变原数组：否

let arr = [1,2,3,4,5]
console.log(arr.slice(1,3))   // [2,3]
console.log(arr)    //  [1,2,3,4,5]

11：arr.forEach(callback) 遍历数组,无 return（callback 的参数： value –当前索引的值，index –索引，array –原数组）

是否改变原数组：foreach 循环遍历该数组是不会改变数组每一项的值的，但是对于引用数据类型的话是会可以改变的

let arr = [1,2,3,4,5]
arr.forEach( (value,index,array)=>{
        console.log(`value:${value}    index:${index}     array:${array}`)
    })
    //  value:1    index:0     array:1,2,3,4,5
    //  value:2    index:1     array:1,2,3,4,5
    //  value:3    index:2     array:1,2,3,4,5
    //  value:4    index:3     array:1,2,3,4,5
    //  value:5    index:4     array:1,2,3,4,5

arr.forEach( (value,index,array)=>{
        value=2
    })
    //  [1,2,3,4,5]

let json = [{name:'zhangsan',age:12}]
json.forEach( (value,index,array)=>{
        value.name='lisi'
    })
    //   {name: 'lisi', age: 12}

12：arr.map(callback) 映射数组(遍历数组),通过 return 返回一个新数组（callback 的参数： value –当前索引的值，index –索引，array –原数组）

是否改变原数组：同 foreach

let arr = [1,2,3,4,5]
arr.map( (value,index,array)=>{
        return value = value * 2

})
console.log(arr)//[2,4,6,8,10]

ps: arr.forEach()和 arr.map()的区别

  1. arr.forEach()是和 for 循环一样，是代替 for。arr.map()是修改数组其中的数据，并返回新的数据。
  2. arr.forEach() 没有 return arr.map() 有 return

13：arr.filter(callback) 过滤数组，返回一个满足要求的数组

是否改变原数组：同 foreach

let arr = [1,2,3,4,5]
let arr1 = arr.filter( (item,index) => item< 3)
console.log(arr1)    // [1, 2]

14：arr.every(callback) 依据判断条件，数组的元素是否全满足，若满足则返回 ture

是否改变原数组：同 foreach

let arr = [1,2,3,4,5]
let arr1 = arr.every( (i, v) => i < 3)
console.log(arr1)    // false
let arr2 = arr.every( (i, v) => i < 10)
console.log(arr2)    // true

15：arr.some() 依据判断条件，数组的元素是否有一个满足，若有一个满足则返回 ture

是否改变原数组：同 foreach

let arr = [1,2,3,4,5]
let arr1 = arr.some( (i, v) => i < 3)
console.log(arr1)    // true
let arr2 = arr.some( (i, v) => i > 10)
console.log(arr2)    // false

16：arr.reduce(callback, initialValue) 迭代数组的所有项，累加器，数组中的每个值（从左到右）合并，最终计算为一个值

是否改变原数组：同 foreach

let arr = [0,1,2,3,4]
let arr1 = arr.reduce((preValue, curValue) =>
    preValue + curValue
)
console.log(arr1)    // 10

17:fill(),数组的填充

是否改变原数组：是

let fruits = ['apple','banana','orange'];
let result = fruits.fill('peal');
//['peal','peal','peal']
常用于自定义测试数据
Array(10).fill("1");

18:find(callback)和 findIndex(callback),前者返回符合条件的第一个值，后者返回第一个值对应的索引

和 filter()方法的区别在于：filter 返回值是所有满足条件的元素组成的数组，
一般在需要使用找到的元素时，用 find()方法

let ages = [1,10,18,19,20]
let result = ages.find((item)=>item>1);//10
let result = ages.findIndex((item)=>item>1);//1

19:includes()和 indexOf(),前者返回 boolean 值，后者返回索引值或-1

let a=[1,2,3];
a.includes(1);//true
a.indexOf(1);//0

总结：会改变原数组的方法：
arr.splice()
arr.reverse()
arr.fill()
arr.copyWithin()
arr.sort()
arr.push()
arr.pop()
arr.unshift()
arr.shift()

不改变数组：
arr.slice()
arr.concat()
arr.join()

视情况而定：
arr.map()
arr.forEach()
arr.every()
arr.some()
arr.filter()
arr.reduce()
arr.entries()
arr.find()
