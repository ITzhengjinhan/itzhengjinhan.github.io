前端常用的居中对齐方式

css居中
css
发布日期:   2020-03-18

一、水平居中的方法

1.行内或类行内元素（比如文本和链接）

在块级父容器中让行内元素居中，只需使用 text-align: center;
这种方法可以让 inline/inline-block/inline-table/inline/flex 等类型的元素实现居中。

<style>
  header, nav {
  text-align: center;
  background: white;
  margin: 20px 0;
  padding: 10px;
}

nav a {
  text-decoration: none;
  background: #333;
  border-radius: 5px;
  color: white;
  padding: 3px 8px;
}
</style>
<header>
  This text is centered.
</header>

<nav role='navigation'>
  <a href="#0">One</a>
  <a href="#0">Two</a>
  <a href="#0">Three</a>
  <a href="#0">Four</a>
</nav>

效果：

2.块级元素

让块级元素居中的方法就是设置 margin-left 和 margin-right 为 auto（前提是已经为元素设置了适当的 width 宽度，
否则块级元素的宽度会被拉伸为父级容器的宽度）。常见用法如下所示：

<style>
  main {
  background: white;
}

.center {
  margin: 0 auto;
  width: 200px;
  background: black;
  padding: 20px;
  color: white;
}
</style>

<main>
  <div class="center">
    I'm a block level element and am centered.
  </div>
</main>

效果：

3.多个块级元素

如果要让多个块级元素在同一水平线上居中，那么可以修改它们的 display 值。这里有两个示例，其中一个使用了 inline-block 的显示方式，
另一个使用了 flexbox 的显示方式：

<style>
main {
  background: white;
}

main div {
  background: black;
  color: white;
  padding: 15px;
  max-width: 125px;
  margin: 5px;
}

.inline-block-center {
  text-align: center;//盒子居中
}
.inline-block-center div {
  display: inline-block;
  text-align: center;//字体居中
}
//方法二
.flex-center {
  display: flex;
  justify-content: center;
}
</style>
<main class="inline-block-center">
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
</main>

<main class="flex-center">
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
</main>

二、垂直居中的方法

1. 可以让 line-height 和 height 相等，从而实现垂直居中

<style>
  main {
  background: white;
}

main div {
  background: black;
  color: white;
  height: 100px;
  line-height: 100px;
  padding: 20px;
  width: 50%;
}
</style>
<main>
  <div>
    I'm a centered line.
  </div>
</main>

效果：

2.多行文本

方法一：对于多行文本，可以通过 CSS 为文本设置一个类似 table-cell 的父级容器，然后使用 vertical-align 属性实现垂直居中
方法二：使用 padding 值

<style>

.center-table {
  display: table;
  height: 250px;
  background: white;
  width: 240px;
}
.center-table p {
  display: table-cell;
  background: black;
  color: white;
  border: 10px solid white;
  vertical-align: middle;
  text-align:center;
}

//方法二
.center-table {
  height: 550px;
  background: white;
  width: 240px;

}
.center-table p {
  padding:30px;
  background: black;
  color: white;
  text-align:center;
}
</style>

<div class="center-table">
  <p>I'm vertically centered multiple lines of text in a CSS-created table layout.</p>
</div>

效果：

3.块级元素

如果我们知道元素的高度，可以这样来实现垂直居中：

.parent {
position: relative;
}
.child {
position: absolute;
top: 50%; height: 100px;
margin-top: -50px;
}

如果我们不知道元素的高度，那么就需要先将元素定位到容器的中心位置，然后使用 transform 的 translate 属性，将元素的中心和父容器的中心重合，
从而实现垂直居中：

.parent {
position: relative;
}
.child {
position: absolute;
top: 50%;
transform:translateY(-50%);
}

4.flexbox

.parent {
display: flex;
flex-direction: column;
justify-content: center;
}

三、水平垂直居中方法

1、宽高不固定元素

设定父级容器为相对定位的容器，设定子元素绝对定位的位置
position: absolute; top: 50%; left: 50%，最后使用负向 margin 实现水平和垂直居中，
margin 的值为（自身宽高）的一半。

<style>
  body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}

main {
  position: relative;
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;

}
main div {
  text-align:center;
  background: black;
  color: white;
  width: 200px;
  height: 100px;
  margin: -50px 0 0 -100px;
  position: absolute;
  top: 50%;
  left: 50%;
}
</style>
<main>
  <div>
     I'm a block-level element a fixed height and width, centered vertically within my parent.
  </div>
</main>

效果：

2. flex

<style>
body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}

main {
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;
  padding: 20px;
  display: flex;
  justify-content: center;
  align-items: center;
  resize: both;
  overflow: auto;
}

main div {
  background: black;
  color: white;
  width: 50%;
  padding: 20px;
  resize: both;
  overflow: auto;
}
</style>
<main>
  <div>
     I'm a block-level-ish element of an unknown width and height, centered vertically within my parent.
  </div>
</main>
